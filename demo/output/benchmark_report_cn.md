## 分析

### 2D 图表

- **柱状图**：ShenBi 与 pyqtgraph 性能持平，远超 matplotlib
- **直方图**：1.4–1.6× 稳定加速，得益于 pyqtgraph 的高效 BarGraphItem
- **折线图**：三方性能接近，pyqtgraph 和 ShenBi 持平，略优 matplotlib
- **散点图**：matplotlib 的 Agg 渲染器在超大散点图上有优势

### 3D 图表

- **3D 散点图**：ShenBi 的 2D 投影方案无需 OpenGL，中等规模表现有竞争力
- **3D 曲面图**：ShenBi contourf 方案在 70×70 网格上比 matplotlib **快 2.5 倍**
